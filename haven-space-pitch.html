System Overview

Haven Space is a Web-Based and Mobile Boarding House Management System that centralizes all boarding house operations into a single platform, connecting the administration office, landlords, and student boarders through role-based dashboards.

Boarders

Browse listings, book rooms, make secure payments, and communicate with landlords

Landlords

Manage properties, screen tenants, track bookings, and monitor income

Admin

Oversee platform operations, manage users, and generate reports

System Features

👤

User Features (Boarders & Landlords)

  • User registration and login with role selection
  • Property listing browsing with search and filters
  • Interactive map view for location-based search
  • Online booking and reservation system
  • In-platform messaging and notifications
  • Secure payment processing with receipts
  • Booking history and favorites management
  • Profile management and document upload
⚙️

Admin Features

  • User management (approve, suspend, verify)
  • Property listing moderation and approval
  • Booking oversight and dispute resolution
  • Payment monitoring and financial reports
  • Platform analytics and usage statistics
  • Content management and announcements
  • System activity logs and audit trails
  • Report generation (PDF/Excel export)
