# 🏠 Haven Space - Presentation Deck

## 1. Key Points About Haven Space

### What is Haven Space?
**Haven Space** is a comprehensive **multi-tenant platform** that connects landlords with boarders through a modern, seamless digital experience—transforming how boarding houses are discovered, managed, and experienced.

### Core Value Proposition
| For Boarders | For Landlords | For Platform Owners |
|--------------|---------------|---------------------|
| Easy discovery & booking | Simplified property management | Revenue from subscriptions & transactions |
| Transparent pricing | Automated payment processing | Scalable SaaS business model |
| Secure payments | Tenant screening tools | Multi-tenant architecture |
| Direct landlord communication | Performance analytics | White-label potential |

### Technology Excellence
- Built on **Next.js 16** with **React 18** & **TypeScript**
- **tRPC** for type-safe, end-to-end communication
- **PostgreSQL** with **Prisma ORM** for robust data management
- **NextAuth.js** with role-based access control
- **Monorepo architecture** (Turborepo + Bun) for maintainability
- **Docker-ready** for seamless deployment

## 4. Business Model Details

### Revenue Streams

| Stream | Description | Pricing Model |
|--------|-------------|---------------|
| **Landlord Subscriptions** | Premium features for landlords | ₱499-₱1,499/month tiers |
| **Transaction Fees** | Payment processing fee | 2-3% per booking |
| **Featured Listings** | Boosted visibility | ₱100-₱500/listing |
| **Verification Services** | ID & property verification | ₱200-₱500 one-time |
| **Enterprise/White-label** | Custom deployments | Custom pricing |

### Subscription Tiers (Landlords)

| Tier | Price | Features |
|------|-------|----------|
| **Free** | ₱0 | 1 property, basic listing, 5% transaction fee |
| **Starter** | ₱499/mo | 5 properties, analytics, 3% transaction fee |
| **Professional** | ₱999/mo | 20 properties, priority support, 2% transaction fee |
| **Enterprise** | ₱1,499/mo | Unlimited properties, API access, 1.5% transaction fee |

### Unit Economics (Projected)
- **Customer Acquisition Cost (CAC)**: ₱500-₱1,000 per landlord
- **Lifetime Value (LTV)**: ₱15,000-₱50,000 per landlord
- **LTV:CAC Ratio**: 15:1 (healthy SaaS metrics)
- **Gross Margin**: 80%+ (software business)
